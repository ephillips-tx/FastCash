FASTCASH: A Coding Project

Count CASH one loop at a time. A javascript coding exercise utilizing loops. Write a program that displays the numbers from 0 to 100. Apend a "$" symbol before each number. Display the even numbers in BOLD.

Instructions

Enter a starting value and an ending value. Press the "COUNT CASH" button and the CASH count will appear.

You can also press the "ReCOUNT" button to reset and count again!
